# LearnHub_Pro

Professional LearnHub scaffold (React + TypeScript + Tailwind + Node + Express + MongoDB)

Quick start:

1. Backend
   cd server
   cp .env.example .env
   npm install
   npm run seed
   npm run dev

2. Frontend
   cd client
   cp .env.example .env
   npm install
   npm run dev

Frontend: http://localhost:5173
Backend: http://localhost:5000/api
